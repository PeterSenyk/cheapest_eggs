About

The Cheapest Egg is a web application we will develop to combat rising grocery prices. Let’s be honest, no one wants to pay $4 for one cucumber. Our application will bring you price data from big box stores, local stores, the random neighborhood farmer too!! If someone is selling something cheap, we want our users to know about it.

How will this be possible? Large retailers have shop flyers and API data that we will import to our web application. As for local stores, this is where you come in! This is why we are different from all other apps.

With your help we want to develop a community driven web application where we, as a community, report the cheapest products in our local stores. Together we will share the cheapest prices with others because we all deserve the cheapest eggs.

Save money, stay organized and go on that vacation you’ve been dreaming about!
